# French real estate viewer

Visualize price per m², rent per m², and gross yield across French municipalities, with yearly breakdowns and evolution tracking from 2020 to 2025.

## Features

- **3 metrics**: Price per m², Rent per m², Gross yield
- **Value / Evolution toggle**: view absolute values or % change over a custom period
- **Filters**: property type (Residential, House, Apartment, Land) and year
- **Municipality search** by name
- CartoDB Voyager GL basemap (MapLibre GL)

## Structure

```
src/
  components/       — React components (MapView, FilterPanel, Search, Legend, LanguageToggle)
  context/          — DataContext (cities, years, computed data)
  locales/          — i18n translation files (fr.json, en.json)
  config.ts         — scale computation, filter fields
  data.ts           — stats, color mapping, tooltip data
  types.ts          — shared TypeScript types
public/
  cities.json       — municipality data generated by process.py
  cities.pmtiles    — vector tiles generated by process.py + tippecanoe
process.py          — DuckDB pipeline (DVF + rents → cities.json + cities.pmtiles)
```

## Setup

```bash
sudo apt install tippecanoe
sudo apt install python3.12-venv
python3 -m venv .venv
.venv/bin/pip install duckdb
```

## Generate data

```bash
.venv/bin/python process.py
```

## Dev

```bash
npm run dev
```
